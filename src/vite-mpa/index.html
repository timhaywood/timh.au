How to automatically add all html pages to your Vite app

By default, Vite assumes you have a single index.html file as an entry point. Any additional html files need to be added manually to your vite.config.js file, as specified here.

To add all your html pages automatically, you can define a glob pattern to return an array of files: 

// vite.config.js          
import { resolve } from "path";
import { defineConfig } from "vite";
import glob from "glob";

export default defineConfig({
  root: "src",
  build: {
    outDir: "../dist",
    rollupOptions: {
      input: glob.sync(
        resolve(__dirname, "src", "**/*.html")
      ),
    },
  },
});

This adds all the .html files in the src directory as pages.

Just make sure to install the glob package using your package manager as well.
